CROWD FUNDING

ABOUT THE PROJECT:
1. Created a Smart Contract named "Fundme.sol" using Solidity.
2. The contract prompts the user to fund "Ethereum" tokens using fund function.
3. The deployment scripts are written using Node.js and Javascript and tested on Hardhat environment.
4. The contract can be made to work in Ethereumm mainnet and other networks by adding
the corresponding RPC_Url and modifying the helper-hardhat-config.js file
5. Both unit testing and stage testing are carried out and verified.
6. The contract is passed to the Etherscan and was verified. It can be checked using the contract address
"0xe7f1725E7734CE288F8367e1Bb143E90bb3F0512"
7. The contract is written in a manner that only the owner can withdraw the funds.
USECASE OF BUTTONS:
1. CONNECT WALLET = Connects to the MetaMask wallet(Reverts if MetaMask is not installed)
2. FUND = Can be used to initiate the funding transaction.
3. BALANCE = On clicking, it shows the total amount of fund being collected in Ethers.
4. WITHDRAW = The funds can be withdrawn by the owner who calls the contract.
